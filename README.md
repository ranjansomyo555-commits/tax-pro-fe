# Tax Pro Solutions Website

Professional website for Tax Pro Solutions - Your trusted tax consultant.

## Setup Instructions

### 1. Add Your Logo
Save your company logo image as `logo.png` in the project folder (c:\Users\ranja\proj1\).

The logo should be:
- Format: PNG, JPG, or SVG
- Recommended size: 150x150 pixels or higher
- File name: `logo.png`

### 2. Open the Website
Simply double-click on `index.html` to open the website in your browser.
